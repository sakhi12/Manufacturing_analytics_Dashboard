# 🏭 Manufacturing Analytics Dashboard

A comprehensive Manufacturing Analytics Dashboard built using **Excel, SQL, Power BI, and Tableau** to monitor production performance, manufacturing efficiency, rejection analysis, and sales KPIs.

---

## 📌 Project Overview

This project provides an end-to-end analytics solution for manufacturing data. It demonstrates how raw production data can be transformed into meaningful business insights using SQL for data analysis and visualization tools like Power BI, Tableau, and Excel.

The dashboard helps manufacturing organizations monitor:

- Production Performance
- Manufacturing Efficiency
- Sales Performance
- Rejection Analysis
- Department Performance
- Machine Performance
- Employee Performance

---

## 🚀 Tech Stack

- Microsoft Excel
- MySQL
- Power BI
- Tableau
